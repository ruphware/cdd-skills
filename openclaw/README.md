# [CDD-8] Master Chef OpenClaw Legacy Path

The canonical `[CDD-8] Master Chef` package root now lives in `../cdd-master-chef/`, and the current OpenClaw adapter docs now live in `../cdd-master-chef/openclaw/`.

Use these paths instead:

- `../cdd-master-chef/SKILL.md`
- `../cdd-master-chef/openclaw/README.md`
- `../cdd-master-chef/openclaw/MASTER-CHEF-RUNBOOK.md`
- `../cdd-master-chef/openclaw/MASTER-CHEF-TEST-HARNESS.md`

This top-level `openclaw/` directory is now a compatibility stub only. It is intentionally not a separate skill package root.
