# cdd-skills

Distribution repo for the CDD (Chat‑Driven Development) Codex skills.

## Install (Codex)

```bash
git clone git@github.com:ruphware/cdd-skills.git ~/Workspace/cdd-skills
cd ~/Workspace/cdd-skills
./scripts/install.sh
```

This copies skills into `~/.agents/skills/`.

## Skills

- `cdd-ship-step`
- `cdd-refresh-index`

Both skills are **explicit-only**; invoke them with `$cdd-ship-step` / `$cdd-refresh-index`.
